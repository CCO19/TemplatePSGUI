# Template GUI PowerShell

Ce script a pour objectif de fournir un template basique pour créer un WinForm professionnel, sans prompt Windows Terminal et avec un icône personnalisé en barre de titre ainsi qu'en barre de tâches.

En commentaire du script, des conseils pour extraire les fichiers d'extesion ".ico" depuis les fichiers d'extensions ".dll" ou ".exe", ou encore une commande PowerShell pour convertir le fichier en Base64 afin de rendre le script autonome pour l'affichage de l'icône.
